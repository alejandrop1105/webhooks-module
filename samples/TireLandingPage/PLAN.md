# Plan de Implementación: Landing Page Neumáticos

## Objetivo
Crear una landing page moderna y premium para una empresa de neumáticos, que redirija a una tienda WooCommerce.

## Estándares Técnicos
- **Framework**: Vite (Vanilla JS) para máximo rendimiento y simplicidad.
- **Estilos**: CSS Puro (Vanilla) con variables CSS para temas (Dark Mode, Colores Vibrantes).
- **Lenguaje**: HTML5 Semántico, JavaScript ES6+.
- **Integración**: Enlaces directos a capacidades de WooCommerce.

## Estructura del Proyecto
- `index.html`: Estructura principal.
- `style.css`: Estilos globales y componentes.
- `main.js`: Lógica de interacción.
- `public/`: Assets estáticos.

## Diseño (UX/UI)
- **Hero Section**: Imagen de alta calidad, CTA a la tienda.
- **Beneficios**: Iconos SVG inline.
- **Categorías**: Grid de productos.
- **Footer**: Contacto.
